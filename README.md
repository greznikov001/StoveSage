StoveSage

AI-driven stovetop monitoring system built as a structured learning program to develop production-grade AI, cloud, and DevOps engineering discipline.

Overview

StoveSage is a mock stovetop hazard monitoring system designed to simulate a real-world AI product while enforcing rigorous engineering standards.

The system captures periodic images of a stovetop, analyzes them using cloud-based AI vision services, and detects persistent cooking hazards such as smoke accumulation, boil-over, sustained flame, overheating, or unattended burners.

When a hazard is detected, the system can sound alarm and send notifications.
